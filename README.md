# LeetCode Hot 100 - C++ 系统学习指南

## 📚 项目简介

本项目是针对 LeetCode Hot 100 题目的系统化学习资源，每道题包含：
- **solution.md**：完整解法、知识点、复杂度分析
- **guide.md**：渐进式编程指导，培养独立解题能力

## 🎯 学习建议

1. **先看 guide.md**：按提示层层深入思考
2. **独立编码**：尝试实现自己的解法
3. **对照 solution.md**：学习优化思路和标准解法
4. **总结知识点**：记录到个人笔记
