# 763. 划分字母区间

## 题目描述

给你一个字符串 `s` 。我们要把这个字符串划分为尽可能多的片段，同一字母最多出现在一个片段中。

注意，划分结果需要满足：将所有划分结果按顺序连接，得到的字符串仍然是 `s` 。

返回一个表示每个字符串片段的长度的列表。

**示例 1：**
```
输入：s = "ababcbacadefegdehijhklij"
输出：[9,7,8]
解释：
划分结果为 "ababcbaca", "defegde", "hijhklij"。
每个字母最多出现在一个片段中。
像 "ababcbacadefegde", "hijhklij" 这样的划分是错误的，因为划分的片段数较少。
```

## 解题思路

### 方法：贪心 + 哈希表
- **核心思想**：如果要划分一个片段，那么这个片段中的任何字母，其在字符串中出现的所有位置都必须包含在这个片段内。换句话说，片段的结束位置至少要是片段中所有字母的最后出现位置的最大值。
- **流程**：
  1. **预处理**：遍历一遍字符串，记录每个字母最后出现的下标 `last[char] = index`。
  2. **贪心划分**：
     - 维护 `start` 和 `end`，表示当前片段的起始和结束位置。
     - 遍历字符串，对于当前字符 `c`，更新 `end = max(end, last[c])`。
     - 如果当前下标 `i == end`，说明找到了一个满足条件的切割点。
       - 记录长度 `end - start + 1`。
       - 更新下一段起始位置 `start = i + 1`。

## 复杂度
- 时间复杂度：O(N)。需要遍历两遍字符串。
- 空间复杂度：O(Σ)，Σ 为字符集大小（26）。
