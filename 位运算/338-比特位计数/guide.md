# 338. 比特位计数

## 题目描述

给你一个整数 `n` ，对于 `0 <= i <= n` 中的每个 `i` ，计算其二进制表示中 **`1` 的个数** 。

**示例 1：**
```
输入：n = 2
输出：[0,1,1]
```

---

## 解法：动态规划 + 位运算

### 核心思路
如果要 O(N) 时间解决，必须利用已经计算过的结果。
观察数字 `x` 和 `x >> 1`（即 `x / 2`）的关系：
- `x >> 1` 会把 `x` 的二进制整体右移一位。
- 如果 `x` 是偶数（末尾是 0），那么 `x` 中 1 的个数和 `x >> 1` 一样。
- 如果 `x` 是奇数（末尾是 1），那么 `x` 中 1 的个数比 `x >> 1` 多 1。

状态转移方程：
`bits[i] = bits[i >> 1] + (i & 1)`

由于 `i >> 1` 一定小于 `i`，所以我们可以从 0 到 n 线性递推。

### 复杂度
- 时间复杂度：O(N)。
- 空间复杂度：O(1)（不计结果数组）。
