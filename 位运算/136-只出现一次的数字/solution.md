# 136. 只出现一次的数字

## 题目描述

给你一个 **非空** 整数数组 `nums` ，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。
你必须设计并实现线性时间复杂度的算法来解决此问题，且该算法只使用常量额外空间。

**示例 1：**
```
输入：nums = [2,2,1]
输出：1
```

**示例 2：**
```
输入：nums = [4,1,2,1,2]
输出：4
```

---

## 解法：位运算（异或）

### 核心思路
这道题是位运算的经典入门题。利用**异或（XOR）**运算的性质：
1. **归零律**：`a ^ a = 0`。两个相同的数异或等于 0。
2. **恒等律**：`a ^ 0 = a`。任何数与 0 异或等于它自己。
3. **交换律和结合律**：`a ^ b ^ c = a ^ c ^ b`。

**应用**：
因为数组中除了一个数只出现一次，其他数都出现两次。
如果我们把数组中所有数都异或起来：
- 出现两次的数会两两抵消变成 0。
- 最后剩下的就是那个只出现一次的数。

例如：`4 ^ 1 ^ 2 ^ 1 ^ 2`
`= 4 ^ (1 ^ 1) ^ (2 ^ 2)`
`= 4 ^ 0 ^ 0`
`= 4`

### 代码实现

```cpp
#include <vector>
using namespace std;

class Solution {
public:
    int singleNumber(vector<int>& nums) {
        int result = 0;
        for (int num : nums) {
            result ^= num;
        }
        return result;
    }
};
```

### 复杂度分析
- **时间复杂度**：O(N)。只需要遍历一次数组。
- **空间复杂度**：O(1)。只用了一个变量。

### 易错点
- 异或运算符是 `^`。

---

## 总结
- 找“唯一出现一次”或“成对出现”的问题，优先考虑异或。
