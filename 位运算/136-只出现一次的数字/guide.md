# 136. 只出现一次的数字

## 题目描述

给你一个 **非空** 整数数组 `nums` ，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。
要求：线性时间复杂度，常量额外空间。

**示例 1：**
```
输入：nums = [2,2,1]
输出：1
```

---

## 解法：异或运算

### 核心思路
题目要求 O(N) 时间和 O(1) 空间，这意味着不能用哈希表（空间 O(N)）或排序（时间 O(N log N)）。
位运算中的**异或（XOR）**完美符合要求。

异或的性质：
1. `x ^ x = 0` （相同的数异或为 0）
2. `x ^ 0 = x` （与 0 异或不变）
3. 满足交换律和结合律。

如果我们把数组里所有的数字都异或起来，成对出现的数字会互相抵消（变成 0），最终剩下的就是那个只出现一次的数字。

### 复杂度
- 时间复杂度：O(N)。
- 空间复杂度：O(1)。
