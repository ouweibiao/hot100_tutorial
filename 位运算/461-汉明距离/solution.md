# 461. 汉明距离

## 题目描述

两个整数之间的 **汉明距离** 指的是这两个数字对应二进制位不同的位置的数目。
给你两个整数 `x` 和 `y`，计算并返回它们之间的汉明距离。

**示例 1：**
```
输入：x = 1, y = 4
输出：2
解释：
1   (0 0 0 1)
4   (0 1 0 0)
       ↑   ↑
上面的箭头指出了对应二进制位不同的位置。
```

---

## 解法：异或 + 统计 1 的个数

### 核心思路
我们要找“不同”的位。
异或运算（`^`）的定义正是：相同为 0，不同为 1。
所以，`x ^ y` 的结果中，二进制位为 1 的位置，就是 `x` 和 `y` 不同的位置。
问题转化为：**计算 `x ^ y` 中 1 的个数**。

我们可以使用：
1. **内置函数**：`__builtin_popcount(x ^ y)`（C++ GCC）。
2. **Brian Kernighan 算法**：不断 `n = n & (n - 1)` 直到为 0。

### 代码实现

```cpp
class Solution {
public:
    int hammingDistance(int x, int y) {
        int xorVal = x ^ y;
        int count = 0;
        
        // 统计 xorVal 中 1 的个数
        while (xorVal > 0) {
            xorVal = xorVal & (xorVal - 1); // 消除最低位的 1
            count++;
        }
        
        return count;
    }
};
```

### 复杂度分析
- **时间复杂度**：O(K)，K 是不同的位数。或者说是 O(log C)，C 是数据范围。
- **空间复杂度**：O(1)。

### 易错点
- 没啥易错点，非常基础。

---

## 总结
- 异或 = 找不同。
- 统计 1 的个数 = Brian Kernighan 算法 (`n & (n-1)`)。
