# 2. 两数相加

## 🧠 解题思路引导

### 1. 模拟加法
- 这道题就像我们在小学做加法算术一样。
- 从最低位（链表头）开始相加。
- 如果和大于等于 10，就要进位（carry）。
- 进位会加到下一位的计算中。

### 2. 细节处理
- **两个链表长度不一样怎么办？**
  - 短的那个链表可以视为后面补了 0。
- **最后如果还有进位怎么办？**
  - 比如 `5 + 5 = 10`，最后需要多加一个节点 `1`。
- **如何优雅地写代码？**
  - 使用 `dummy` 节点（哑节点）作为结果链表的头，方便操作。
  - 使用一个 `while` 循环，只要 `l1` 或 `l2` 还有节点，或者 `carry` 不为 0，就继续。

### 3. 为什么是逆序存储？
- 题目说数字是逆序存储的，这其实简化了问题。
- 因为加法就是从低位开始算的，正好对应链表的头部。
- 如果是正序存储（高位在头），那还得先反转链表或者用栈，反而麻烦。
