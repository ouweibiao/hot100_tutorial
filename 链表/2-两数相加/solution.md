# 2. 两数相加

## 题目描述

给你两个 **非空** 的链表，表示两个非负的整数。它们每位数字都是按照 **逆序** 的方式存储的，并且每个节点只能存储 **一位** 数字。

请你将两个数相加，并以相同形式返回一个表示和的链表。

你可以假设除了数字 0 之外，这两个数都不会以 0 开头。

**示例 1：**
```
输入：l1 = [2,4,3], l2 = [5,6,4]
输出：[7,0,8]
解释：342 + 465 = 807.
```

**示例 2：**
```
输入：l1 = [0], l2 = [0]
输出：[0]
```

**示例 3：**
```
输入：l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
输出：[8,9,9,9,0,0,0,1]
```

---

## 解法：模拟加法

### 核心思路
1. 同时遍历两个链表，逐位计算 `sum = val1 + val2 + carry`。
2. 当前位的值为 `sum % 10`。
3. 新的进位为 `sum / 10`。
4. 如果一个链表遍历完了，就当它的值为 0。
5. 循环结束时，如果 `carry > 0`，还需要追加一个进位节点。

### 代码实现
```cpp
class Solution {
public:
    ListNode* addTwoNumbers(ListNode* l1, ListNode* l2) {
        ListNode dummy(0); // 哑节点
        ListNode* tail = &dummy;
        int carry = 0;
        
        // 只要还有一个链表没走完，或者还有进位，就继续
        while (l1 != nullptr || l2 != nullptr || carry != 0) {
            int val1 = (l1 != nullptr) ? l1->val : 0;
            int val2 = (l2 != nullptr) ? l2->val : 0;
            
            int sum = val1 + val2 + carry;
            
            // 更新进位
            carry = sum / 10;
            
            // 创建新节点
            tail->next = new ListNode(sum % 10);
            tail = tail->next;
            
            // 移动指针
            if (l1 != nullptr) l1 = l1->next;
            if (l2 != nullptr) l2 = l2->next;
        }
        
        return dummy.next;
    }
};
```

### 复杂度分析
- **时间复杂度**：O(max(m, n))，需要遍历较长的那个链表。
- **空间复杂度**：O(1)，返回值不计入空间复杂度（如果是纯粹的算法空间），或者 O(max(m, n)) 如果算上结果链表。

### 易错点
1. **忘记处理进位**：特别是最后一位相加产生进位的情况（如 `5 + 5 = 10`）。
2. **指针判空**：链表长度不一致时，短链表的指针会先变成 `nullptr`，访问 `val` 前必须检查。

## 总结

- 这是一个非常标准的**链表模拟题**。
- `while (l1 || l2 || carry)` 这个循环条件是写出简洁代码的关键。
