# 234. 回文链表

## 🧠 解题思路引导

### 1. 什么是回文？
- 1 -> 2 -> 2 -> 1 (是)
- 1 -> 2 -> 3 (否)
- 正着读和反着读一样。

### 2. 简单的做法（空间换时间）
- 链表只能单向遍历，能不能把它变成数组？
- 如果复制到 `vector<int>` 中，用双指针判断回文数组就很简单了。
- 空间复杂度是 O(n)，题目进阶要求是 O(1)，怎么办？

### 3. 进阶思考（O(1) 空间）
- 我们能不能只反转链表的一半？
- 如果把后半部分反转，比如 `1 -> 2 -> 3 -> 2 -> 1` 变成：
  - 前半部分：`1 -> 2`
  - 后半部分（反转后）：`1 -> 2`
- 这样就可以同时遍历比较了！

### 4. 步骤拆解
1. 找到链表中点（快慢指针）。
2. 反转后半部分链表。
3. 比较两部分是否相同。
4. (可选) 恢复链表结构。
