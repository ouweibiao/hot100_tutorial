# 142. 环形链表 II

## 🧠 解题思路引导

### 1. 回顾：判断是否有环
- 在 [141. 环形链表] 中，我们用快慢指针判断是否有环。
- `fast` 每次走两步，`slow` 每次走一步。
- 如果相遇，说明有环。

### 2. 进阶：寻找入环点
- 仅仅知道有环是不够的，我们需要找到**刚进入环的那个节点**。
- 当快慢指针相遇时，它们在环里的某个位置，但不一定是起点。
- 这是一个纯数学问题。

### 3. 数学推导（不要怕！）
- 假设：
  - 链表头到环入口距离为 `x`。
  - 环入口到相遇点距离为 `y`。
  - 相遇点回到环入口距离为 `z`。
- 相遇时：
  - `slow` 走的距离：`x + y`
  - `fast` 走的距离：`x + y + n * (y + z)` (多走了 n 圈)
- 关键等式：
  - `fast` 速度是 `slow` 的 2 倍。
  - `2 * (x + y) = x + y + n * (y + z)`
  - 化简得：`x = (n - 1) * (y + z) + z`
- 结论：
  - 当 n = 1 时，`x = z`。
  - 这意味着：**从头结点出发一个指针，从相遇点出发一个指针，它们会在环入口相遇！**
