# 160. 相交链表

## 🧠 解题思路引导

### 1. 初步思考
- 如果两个链表相交，它们的结构是怎样的？（呈现 Y 字形）
- 相交点之后的节点对于两个链表来说是共享的吗？
- 如果两个链表长度不一样，怎么让它们"同步"到达终点？

### 2. 暴力法
- 对于链表A的每个节点，去遍历链表B，看是否相等。
- 复杂度是多少？能接受吗？

### 3. 哈希表法
- 如果我们可以记住链表A的所有节点...
- 遍历链表A，把所有节点地址存入哈希表。
- 遍历链表B，检查当前节点是否在哈希表中。
- 空间复杂度是多少？

### 4. 双指针法（巧妙解法）
- 假设链表A长度为 `a`，链表B长度为 `b`，公共部分长度为 `c`。
- `a + (b - c) = b + (a - c)`
- 如果两个指针分别走 A -> B 和 B -> A，它们会相遇吗？
- 走过的总路程是多少？
