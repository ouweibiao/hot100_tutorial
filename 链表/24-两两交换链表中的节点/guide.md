# 24. 两两交换链表中的节点

## 🧠 解题思路引导

### 1. 题目分析
- 给定 `1 -> 2 -> 3 -> 4`
- 变成 `2 -> 1 -> 4 -> 3`
- 也就是每两个节点交换一下位置。
- **限制**：不能只是交换节点内部的值（val），必须真正交换节点（修改指针）。

### 2. 递归法（Top-Down）
- 想象一下，对于 `head` (1) 和 `head->next` (2)：
- 交换后，(2) 变成了新的头，(1) 变成了第二个节点。
- (1) 的 `next` 应该指向哪里？应该指向 **后面剩余部分交换后的结果**。
- 这就是典型的递归结构！
- `swap(1, 2, 3, 4) = 2 -> 1 -> swap(3, 4)`

### 3. 迭代法（Bottom-Up）
- 如果用循环怎么做？
- 我们需要维护三个指针：`prev`（前驱），`curr`（当前），`next`（下一个）。
- 每次操作：
  - `prev` 指向 `next`
  - `curr` 指向 `next->next`
  - `next` 指向 `curr`
- 这里指针操作非常容易晕，**画图**是最好的办法。
- 记得用 `dummy` 节点来处理头部交换。
