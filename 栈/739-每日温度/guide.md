# 739. 每日温度

## 题目描述

给定一个整数数组 `temperatures` ，表示每天的温度，返回一个数组 `answer` ，其中 `answer[i]` 是指对于第 `i` 天，下一个更高温度出现在几天后。如果气温在这之后都不会升高，请在该位置用 `0` 来代替。

**示例 1:**
```
输入: temperatures = [73,74,75,71,69,72,76,73]
输出: [1,1,4,2,1,1,0,0]
```

## 解题思路

### 方法一：暴力法
- 两层循环，对于每个 `i`，向后找第一个比它大的 `j`。
- 时间复杂度：O(N²)。数据量大时会超时。

### 方法二：单调栈 (Monotonic Stack) —— 核心必考
- **场景**：寻找数组中**右边第一个比当前元素大（或小）的元素**。
- **原理**：
  - 维护一个**单调递减**的栈。栈中存放**下标**。
  - 遍历数组：
    - 如果当前温度 `T[i]` <= 栈顶温度 `T[top]`：直接入栈（保持递减）。
    - 如果当前温度 `T[i]` > 栈顶温度 `T[top]`：
      - 说明我们找到了栈顶元素的"下一个更高温度"！
      - 弹出栈顶 `idx = stack.pop()`。
      - 记录答案：`ans[idx] = i - idx`。
      - 继续比较新的栈顶，直到栈为空或当前温度不再大于栈顶。
      - 将当前 `i` 入栈。

### 图解
`[73, 74, ...]`
1. 栈空，push 0 (73)。Stack: [0]
2. i=1 (74)。74 > 73。Pop 0。Ans[0] = 1 - 0 = 1。Push 1 (74)。Stack: [1]
...

## 复杂度
- **时间复杂度**：O(N)。每个元素最多入栈一次，出栈一次。
- **空间复杂度**：O(N)。
