# 84. 柱状图中最大的矩形

## 题目描述

给定 `n` 个非负整数，用来表示柱状图中各个柱子的高度。每个柱子彼此相邻，且宽度为 1 。求在该柱状图中，能够勾勒出来的矩形的最大面积。

**示例 1:**
```
输入：heights = [2,1,5,6,2,3]
输出：10
解释：最大的矩形为 5 和 6 组成的区域，高度为 5，宽度为 2，面积 = 10。
```

## 解题思路

### 方法一：暴力枚举 (超时)
- 枚举每根柱子 `i` 作为高度，向左右扩展找到边界。
- 时间复杂度 O(N²)。

### 方法二：单调栈 (Hard) —— 进阶必考
- **核心思想**：对于每根柱子 `heights[i]`，如果我们能找到它**左边第一个比它小**的位置 `left`，和**右边第一个比它小**的位置 `right`，那么以 `heights[i]` 为高的矩形宽度就是 `(right - left - 1)`。
- **转化**：这就变成了"寻找左右两边第一个更小元素"的问题，正是单调栈的拿手好戏。
- **栈的性质**：单调递增栈（存下标）。
  - 当遇到一个比栈顶小的元素 `h[i]` 时，说明栈顶元素的**右边界**确定了（就是 `i`）。
  - 栈顶元素的**左边界**就是它在栈内的下一个元素（因为栈是递增的）。
  - 此时可以弹出栈顶，计算面积。

### 哨兵技巧 (Sentinel)
- 为了避免处理栈空和最后还有残留元素的情况，通常在数组**首尾**各加一个高度为 0 的哨兵。
  - 首部 0：保证栈永远不空（至少有个 0）。
  - 尾部 0：保证原数组所有元素最终都能出栈并计算面积。

## 复杂度
- **时间复杂度**：O(N)。
- **空间复杂度**：O(N)。
