# 415. 字符串相加

## 题目描述

给定两个字符串形式的非负整数 `num1` 和 `num2` ，计算它们的和并同样以字符串形式返回。

你不能使用任何內建的用于处理大整数的库（比如 `BigInteger`）， 也不能直接将输入的字符串转换为整数形式。

**示例 1：**
```
输入：num1 = "11", num2 = "123"
输出："134"
```

## 解题思路

### 方法：模拟竖式加法
这是一道非常经典的考察编程基本功的题目。核心就是模拟我们在纸上算加法的过程。

1. **双指针**：`i` 指向 `num1` 末尾，`j` 指向 `num2` 末尾。
2. **进位 `carry`**：记录上一位的进位值，初始为 0。
3. **循环**：只要 `i >= 0` 或 `j >= 0` 或 `carry != 0`，就继续计算。
   - 取值：如果 `i >= 0`，取 `num1[i] - '0'`，否则取 0。同理取 `num2[j]`。
   - 求和：`sum = x + y + carry`。
   - 更新结果：`sum % 10` 追加到字符串。
   - 更新进位：`carry = sum / 10`。
   - 移动指针：`i--`, `j--`。
4. **反转**：因为是从低位算到高位（从后往前），结果是反的，最后需要 `reverse` 一下。

## 复杂度
- **时间复杂度**：O(max(M, N))，其中 M, N 是两个字符串长度。
- **空间复杂度**：O(1)（不计结果存储）。
