# 647. 回文子串

## 题目描述

给你一个字符串 `s` ，请你统计并返回这个字符串中 **回文子串** 的数目。

**示例 1：**
```
输入：s = "abc"
输出：3
解释：三个回文子串: "a", "b", "c"
```

**示例 2：**
```
输入：s = "aaa"
输出：6
解释：6个回文子串: "a", "a", "a", "aa", "aa", "aaa"
```

## 解题思路

### 方法一：中心扩展法 (Center Expansion)
这道题与 **5. 最长回文子串** 几乎一模一样，只是从"找最长"变成了"数个数"。

- **思路**：
  - 枚举所有的回文中心（共 2n - 1 个）。
  - 从中心向外扩展，只要是回文串，计数器就加 1。
  - 继续扩展，直到不满足回文条件。
- **复杂度**：O(n²)。

### 方法二：动态规划
- `dp[i][j]` 表示 `s[i..j]` 是否为回文串。
- 状态转移方程：
  - 如果 `s[i] == s[j]`：
    - 如果 `j - i <= 1` (即长度为 1 或 2)，则 `dp[i][j] = true`。
    - 如果 `j - i > 1`，则 `dp[i][j] = dp[i+1][j-1]`。
  - 如果 `s[i] != s[j]`，则 `dp[i][j] = false`。
- 在填表的过程中统计 `true` 的个数。
- **注意**：填表顺序需要从下往上（i 从大到小），从左往右（j 从小到大）。

## 总结
依然首选 **中心扩展法**，因为它不需要额外的 O(n²) 空间来存储 DP 表。
