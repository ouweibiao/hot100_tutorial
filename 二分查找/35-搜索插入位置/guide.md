# 35. 搜索插入位置

## 题目描述

给定一个排序数组和一个目标值，在数组中找到目标值，并返回其索引。如果目标值不存在于数组中，返回它将会被按顺序插入的位置。

请必须使用时间复杂度为 `O(log n)` 的算法。

**示例 1:**
```
输入: nums = [1,3,5,6], target = 5
输出: 2
```

## 解题思路

### 方法：二分查找
- **核心思想**：
  - 标准的二分查找。
  - 我们要找的是第一个 `>= target` 的元素下标。
- **流程**：
  - 初始化 `left = 0`, `right = n - 1`。
  - 当 `left <= right` 时：
    - `mid = left + (right - left) / 2`。
    - 如果 `nums[mid] >= target`，说明答案在 `mid` 左边（包含 `mid`），`right = mid - 1`，并记录 `ans = mid`（或最后返回 `left`）。
    - 如果 `nums[mid] < target`，说明答案在 `mid` 右边，`left = mid + 1`。
  - 最后返回 `left`。因为循环结束时，`left` 指向第一个大于等于 `target` 的位置。

## 复杂度
- 时间复杂度：O(log N)。
- 空间复杂度：O(1)。
