# 51. N 皇后

## 题目描述

按照国际象棋的规则，皇后可以攻击与之处在同一行、同一列或同一条斜线上的棋子。

**n 皇后问题** 研究的是如何将 `n` 个皇后放置在 `n×n` 的棋盘上，并且使皇后彼此之间不能相互攻击。

给你一个整数 `n` ，返回所有不同的 **n 皇后问题** 的解决方案。

## 解题思路

### 方法：回溯 (Backtracking)
- **核心**：逐行放置皇后。
- **限制条件**：
  1. 不能同行（因为我们是逐行放置的，天然满足）。
  2. 不能同列：使用一个数组或哈希集合 `cols` 记录哪一列已有皇后。
  3. 不能在主对角线（左上到右下）：对于坐标 `(row, col)`，`row - col` 是常数。
  4. 不能在副对角线（右上到左下）：对于坐标 `(row, col)`，`row + col` 是常数。
- **流程**：
  - `backtrack(row)`
  - 遍历 `col` 从 0 到 n-1：
    - 检查 `col`、`row-col`、`row+col` 是否被占用。
    - 如果未被占用：
      - 放置皇后（更新棋盘和占用集合）。
      - 递归 `backtrack(row + 1)`。
      - 回溯（撤销放置）。

## 复杂度
- 时间复杂度：O(N!)。
- 空间复杂度：O(N)。
