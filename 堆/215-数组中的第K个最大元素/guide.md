# 215. 数组中的第K个最大元素

## 题目描述

给定整数数组 `nums` 和整数 `k`，请返回数组中第 `k` 个最大的元素。

**示例 1:**
```
输入: [3,2,1,5,6,4], k = 2
输出: 5
```

---

## 解法：堆排序 / 快速选择

### 核心思路
这是一个经典的 **Top K** 问题。主要有两种解法：

#### 1. 优先队列（堆）
我们可以维护一个大小为 `k` 的**小顶堆**。
当堆中元素数量小于 `k` 时，直接添加。
当堆满时，如果新元素大于堆顶（堆中最小值），则弹出堆顶，放入新元素。
这样，堆中始终保留着当前遍历过的所有元素中**最大的 k 个**。
最后，堆顶元素就是这 k 个里面最小的，也就是**第 k 大**的元素。

#### 2. 快速选择 (Quick Select)
基于快速排序的 Partition 操作。
每次 Partition 会将一个元素放到它最终排序后的位置 `p`。
- 如果 `p` 正好是倒数第 `k` 个位置，直接返回。
- 如果 `p` 在目标位置左边，则递归右半部分。
- 如果 `p` 在目标位置右边，则递归左半部分。
这种方法平均时间复杂度为 O(N)。

### 复杂度
- **堆**：时间 O(N log K)，空间 O(K)。
- **快速选择**：平均时间 O(N)，空间 O(log N)（递归栈）。
