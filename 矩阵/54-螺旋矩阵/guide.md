# 54. 螺旋矩阵

## 题目描述

给你一个 `m` 行 `n` 列的矩阵 `matrix` ，请按照 **顺时针螺旋顺序** ，返回矩阵中的所有元素。

**示例 1：**
```
输入：matrix = [[1,2,3],[4,5,6],[7,8,9]]
输出：[1,2,3,6,9,8,7,4,5]
```

---

## 解法：模拟 / 边界控制

### 核心思路
我们可以定义四个方向的边界：
- `top`: 上边界
- `bottom`: 下边界
- `left`: 左边界
- `right`: 右边界

然后按照 **右 -> 下 -> 左 -> 上** 的顺序循环打印。
每打印完一条边，就将对应的边界向内收缩一格。
例如，打印完上边（从左到右），`top` 就加 1。
如果在收缩后，发现边界交错（例如 `top > bottom`），说明遍历结束。

### 复杂度
- 时间复杂度：O(MN)。
- 空间复杂度：O(1)。
