# 240. 搜索二维矩阵 II

## 题目描述

编写一个高效的算法来搜索 `m x n` 矩阵 `matrix` 中的一个目标值 `target` 。该矩阵具有以下特性：
- 每行的元素从左到右升序排列。
- 每列的元素从上到下升序排列。

**示例 1：**
```
输入：matrix = [[1,4,7,11,15],[2,5,8,12,19],[3,6,9,16,22],[10,13,14,17,24],[18,21,23,26,30]], target = 5
输出：true
```

---

## 解法：Z 字形查找（从右上角出发）

### 核心思路
由于矩阵的行和列都是有序的，我们可以利用这个性质进行**剪枝**。
选取**右上角** `(0, n-1)` 作为起点：
1. 如果当前元素 **大于** 目标值：
   - 因为这一列是递增的，下面的数只会更大。
   - 所以**这一列**都不可能有目标值，向左移动一列。
2. 如果当前元素 **小于** 目标值：
   - 因为这一行是递增的，左边的数只会更小。
   - 所以**这一行**都不可能有目标值，向下移动一行。
3. 如果相等，则找到。

这样，每次比较都可以排除一行或一列，路径呈 Z 字形（或阶梯状）。

### 复杂度
- 时间复杂度：O(M + N)。最多移动 M 行 N 列。
- 空间复杂度：O(1)。
