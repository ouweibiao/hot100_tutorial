# 49. 字母异位词分组

## 🧠 解题思路引导

### 1. 什么是"字母异位词"？
- 比如 `eat`, `tea`, `ate`。
- 它们的特点是：**字母相同，但排列不同**。
- 这意味着，如果我们把它们按照字母序排序，它们都会变成 `aet`。

### 2. 如何分组？
- 既然排序后它们都一样，那我们可以用**排序后的字符串**作为 key。
- 把所有原始字符串放到这个 key 对应的列表中。
- 例如：
  - key: "aet" -> value: ["eat", "tea", "ate"]
  - key: "ant" -> value: ["tan", "nat"]
  - key: "abt" -> value: ["bat"]

### 3. 数据结构选择
- 我们需要一个映射关系：`string -> list of strings`。
- 在 C++ 中，就是 `unordered_map<string, vector<string>>`。

### 4. 优化思考（计数法）
- 排序的时间复杂度是 `O(k log k)`，其中 k 是字符串长度。
- 如果字符串很长，排序会慢。
- 另一种方法是统计每个字母出现的次数。
- 比如 `eat` -> `a:1, e:1, t:1, 其他:0`。
- 我们可以把这个计数结果编码成一个字符串（比如 `a1e1t1` 或者直接用 26 个数字的序列）作为 key。
- 这种方法的 key 生成复杂度是 `O(k)`。
