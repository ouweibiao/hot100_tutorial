# 108. 将有序数组转换为二叉搜索树

## 题目描述

给你一个整数数组 `nums` ，其中元素已经按 **升序** 排列，请你将其转换为一棵 **高度平衡** 二叉搜索树。

**高度平衡** 二叉树是一棵满足「每个节点的左右两个子树的高度差的绝对值不超过 1 」的二叉树。

## 解题思路

### 方法：递归 / 分治
- **核心思想**：为了保证高度平衡，我们应该总是选择数组的 **中间元素** 作为根节点。
- **流程**：
  1. 找到数组中间位置 `mid`。
  2. 创建根节点 `root`，值为 `nums[mid]`。
  3. 递归构造左子树：范围是 `[left, mid - 1]`。
  4. 递归构造右子树：范围是 `[mid + 1, right]`。
  5. 返回 `root`。
- **为什么平衡？**
  - 因为每次都取中间，左右两边的元素数量差异不超过 1，所以构建出来的树高度差也不会超过 1。

## 复杂度
- 时间复杂度：O(N)。
- 空间复杂度：O(log N) (递归栈深度)。
