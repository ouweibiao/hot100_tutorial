# 101. 对称二叉树

## 题目描述

给你一个二叉树的根节点 `root` ， 检查它是否轴对称。

**示例 1:**
```
    1
   / \
  2   2
 / \ / \
3  4 4  3
```
输入：root = [1,2,2,3,4,4,3]
输出：true

## 解题思路

### 方法一：递归 (DFS)
- **核心思想**：判断两个子树（左子树 `left` 和右子树 `right`）是否镜像对称。
- **终止条件**：
  - 两个节点都为空 -> true。
  - 一个为空一个不为空 -> false。
  - 两个节点值不相等 -> false。
- **递归过程**：
  - 判断 `left.val == right.val`。
  - 判断 `left.left` 和 `right.right` 是否对称（外侧）。
  - 判断 `left.right` 和 `right.left` 是否对称（内侧）。
  - 只有上述条件都满足，才对称。

### 方法二：迭代 (BFS)
- 使用队列，每次将成对的节点入队（左的左和右的右，左的右和右的左）。
- 每次取出两个节点比较，如果有差异则返回 false。

## 推荐
递归写法非常简洁，是本题的标准解法。
