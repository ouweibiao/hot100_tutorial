# 543. 二叉树的直径

## 题目描述

给你一棵二叉树的根节点，返回该树的 **直径** 。

二叉树的 **直径** 是指树中任意两个节点之间最长路径的 **长度** 。这条路径可能经过也可能不经过根节点。

**注意**：两节点之间的路径长度是以它们之间边的数目表示。

## 解题思路

### 方法：深度优先搜索 (DFS)
- **关键点**：
  - 直径 = 左子树深度 + 右子树深度。
  - 但路径不一定经过根节点，所以我们需要在遍历每个节点时，都计算以该节点为"拐点"的路径长度（左深度+右深度），并更新全局最大值 `ans`。
- **函数定义**：`depth(node)` 返回以 `node` 为根的子树的最大深度。
  - `depth(node) = max(depth(node->left), depth(node->right)) + 1`。
- **在递归过程中更新答案**：
  - `ans = max(ans, left_depth + right_depth)`。
- **返回值**：
  - 返回当前节点的深度，供父节点使用。

## 易错点
- 题目求的是"边的数目"，也就是路径上的节点数减 1。如果我们的深度定义为节点数（叶子节点深度为1），那么路径长度就是 `L + R`。
- 最终结果是全局最大值 `ans`，而不是根节点的 `L+R`。
