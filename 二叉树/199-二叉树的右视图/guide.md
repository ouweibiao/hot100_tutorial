# 199. 二叉树的右视图

## 题目描述

给定一个二叉树的 **根节点** `root`，想象自己站在它的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。

**示例 1:**
```
   1            <---
 /   \
2     3         <---
 \     \
  5     4       <---
```
输出: [1,3,4]

## 解题思路

### 方法一：BFS (层序遍历)
- **核心**：每层最后一个被遍历到的节点，就是右视图看到的节点。
- **做法**：
  - 标准层序遍历。
  - 在每一层的循环中，当 `i == currentLevelSize - 1` 时，将该节点值加入结果。

### 方法二：DFS (根-右-左)
- **核心**：我们优先访问右子树。
- **做法**：
  - 按照 **根 -> 右 -> 左** 的顺序遍历。
  - 记录当前遍历的深度 `depth`。
  - 如果当前深度 `depth` 等于结果数组的大小（说明这一层是第一次访问），将当前节点加入结果。
  - 因为优先访问右边，所以每一层第一个被访问的节点一定是该层最右边的节点。

## 推荐
BFS 方法直观且容易理解，推荐。
