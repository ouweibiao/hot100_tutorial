# 236. 二叉树的最近公共祖先

## 题目描述

给定一个二叉树, 找到该树中两个指定节点的最近公共祖先。

**百度百科**中最近公共祖先的定义为：“对于有根树 T 的两个节点 p、q，最近公共祖先表示为一个节点 x，满足 x 是 p、q 的祖先且 x 的深度尽可能大（**一个节点也可以是它自己的祖先**）。”

## 解题思路

### 方法：递归 (后序遍历)
- **核心逻辑**：
  - 如果当前节点 `root` 等于 `p` 或 `q`，或者 `root` 为空，直接返回 `root`。
  - 递归去左子树找 `p` 或 `q`，结果记为 `left`。
  - 递归去右子树找 `p` 或 `q`，结果记为 `right`。
  - **判断**：
    1. 如果 `left` 和 `right` 都不为空，说明 `p` 和 `q` 分别在 `root` 的两侧，那么 `root` 就是最近公共祖先。
    2. 如果只有 `left` 不为空，说明 `p` 和 `q` 都在左子树（或者其中一个是另一个的祖先且在左边），返回 `left`。
    3. 如果只有 `right` 不为空，同理返回 `right`。
    4. 如果都为空，返回 `null`。

## 复杂度
- 时间复杂度：O(N)。
- 空间复杂度：O(N) (递归栈)。
