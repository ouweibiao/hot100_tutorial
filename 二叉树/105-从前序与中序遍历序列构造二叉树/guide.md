# 105. 从前序与中序遍历序列构造二叉树

## 题目描述

给定两个整数数组 `preorder` 和 `inorder` ，其中 `preorder` 是二叉树的先序遍历， `inorder` 是同一棵树的中序遍历，请构造二叉树并返回其根节点。

**示例 1:**
```
输入: preorder = [3,9,20,15,7], inorder = [9,3,15,20,7]
输出: [3,9,20,null,null,15,7]
```

## 解题思路

### 方法：递归 / 分治
- **核心规律**：
  - **前序遍历**：`[根节点 | 左子树 | 右子树]`。第一个元素一定是根节点。
  - **中序遍历**：`[左子树 | 根节点 | 右子树]`。根节点把数组分成左右两部分。
- **算法流程**：
  1. 取出 `preorder` 的第一个元素 `rootVal` 作为根节点。
  2. 在 `inorder` 中找到 `rootVal` 的位置 `index`。
  3. 根据 `index`，计算左子树的节点数量 `leftSize = index - inStart`。
  4. 递归构造左子树：
     - 前序范围：`[preStart + 1, preStart + leftSize]`
     - 中序范围：`[inStart, index - 1]`
  5. 递归构造右子树：
     - 前序范围：`[preStart + leftSize + 1, preEnd]`
     - 中序范围：`[index + 1, inEnd]`
- **优化**：使用哈希表存储 `inorder` 中元素到索引的映射，从而 O(1) 找到根节点位置。

## 复杂度
- 时间复杂度：O(N)。
- 空间复杂度：O(N) (哈希表 + 递归栈)。
