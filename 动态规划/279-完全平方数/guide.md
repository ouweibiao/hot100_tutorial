# 279. 完全平方数

## 题目描述

给你一个整数 `n` ，返回和为 `n` 的完全平方数的最少数量。

完全平方数 是一个整数，其值等于另一个整数的平方；换句话说，其值等于一个整数自乘的积。例如，1、4、9 和 16 都是完全平方数，而 3 和 11 不是。

**示例 1:**
```
输入：n = 12
输出：3
解释：12 = 4 + 4 + 4
```

## 解题思路

### 方法：动态规划 (完全背包)
- **问题转化**：
  - 背包容量为 `n`。
  - 物品为 `1, 4, 9, 16, ...` (所有小于等于 n 的完全平方数)。
  - 每个物品可以无限次使用。
  - 目标：求凑满背包所需的最少物品数量。
- **状态定义**：
  - `dp[i]` 表示和为 `i` 的完全平方数的最少数量。
- **初始化**：
  - `dp[0] = 0`。
  - 其他 `dp[i] = INT_MAX`。
- **状态转移**：
  - `dp[i] = min(dp[i], dp[i - k] + 1)`，其中 `k` 是完全平方数。
  - 遍历顺序：先遍历物品（完全平方数）或先遍历容量都可以，这里习惯先遍历容量。

### 其他方法
- **BFS (广度优先搜索)**：将问题看作图的最短路径问题，从 `n` 到 `0`，每次减去一个完全平方数算一步。
- **数学 (四平方和定理)**：任何正整数都可以表示为至多 4 个平方数之和。结果只能是 1, 2, 3, 4。

## 推荐
DP 方法最通用，适合面试。
