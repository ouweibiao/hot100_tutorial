# 416. 分割等和子集

## 题目描述

给你一个 **只包含正整数** 的非空数组 `nums` 。请你判断是否可以将这个数组分割成两个子集，使得两个子集的元素和相等。

**示例 1:**
```
输入：nums = [1,5,11,5]
输出：true
解释：数组可以分割成 [1, 5, 5] 和 [11] 。
```

## 解题思路

### 方法：动态规划 (0-1 背包问题)
- **问题转化**：
  - 如果数组总和 `sum` 是奇数，直接返回 `false`。
  - 如果是偶数，问题转化为：**能否从数组中选出一些数，使得它们的和恰好等于 `target = sum / 2`**。
  - 这就是一个典型的 **0-1 背包问题**：背包容量为 `target`，物品重量和价值都是 `nums[i]`。
- **状态定义**：
  - `dp[j]` 表示容量为 `j` 的背包能否刚好装满（即和是否能恰好为 `j`）。类型为 `bool`。
- **状态转移**：
  - `dp[j] = dp[j] || dp[j - nums[i]]`
  - 解释：如果不选 `nums[i]`，能否凑出 `j` 取决于之前的状态 `dp[j]`；如果选 `nums[i]`，取决于 `dp[j - nums[i]]`。
- **遍历顺序**：
  - 外层遍历物品 `nums[i]`。
  - 内层遍历容量 `j`，**从大到小**（防止同一个物品被使用多次，这是滚动数组优化 0-1 背包的标准做法）。

## 复杂度
- 时间复杂度：O(N * Target)，其中 Target 是 sum/2。
- 空间复杂度：O(Target)。
