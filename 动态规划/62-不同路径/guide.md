# 62. 不同路径

## 题目描述

一个机器人位于一个 `m x n` 网格的左上角 （起始点在下图中标记为 “Start” ）。
机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（在下图中标记为 “Finish” ）。
问总共有多少条不同的路径？

**示例 1:**
```
输入：m = 3, n = 7
输出：28
```

## 解题思路

### 方法一：动态规划
- **状态定义**：`dp[i][j]` 表示到达 `(i, j)` 的路径数量。
- **状态转移**：
  - 机器人只能从上面 `(i-1, j)` 或左边 `(i, j-1)` 过来。
  - `dp[i][j] = dp[i-1][j] + dp[i][j-1]`。
- **初始化**：
  - 第一行和第一列的所有位置，路径数都为 1（因为只能一直向右或一直向下）。
- **空间优化**：
  - 只需要上一行的信息，可以将二维数组优化为一维数组 `dp[j]`。
  - `dp[j] = dp[j] + dp[j-1]` （新 `dp[j]` = 旧 `dp[j]` + 新 `dp[j-1]`）。

### 方法二：数学 (组合数)
- 总共需要走 `m + n - 2` 步。
- 其中向下走 `m - 1` 步，向右走 `n - 1` 步。
- 答案即为组合数 `C(m + n - 2, m - 1)` 或 `C(m + n - 2, n - 1)`。
