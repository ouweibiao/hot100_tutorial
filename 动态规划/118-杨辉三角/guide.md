# 118. 杨辉三角

## 题目描述

给定一个非负整数 `numRows`，生成「杨辉三角」的前 `numRows` 行。

在杨辉三角中，每个数是它左上方和右上方的数的和。

**示例 1:**
```
输入: numRows = 5
输出: [[1],[1,1],[1,2,1],[1,3,3,1],[1,4,6,4,1]]
```

## 解题思路

### 方法：动态规划 / 模拟
- **状态定义**：`dp[i][j]` 表示第 `i` 行第 `j` 个数。
- **状态转移**：
  - 每行第一个数和最后一个数都是 1。
  - 中间的数：`dp[i][j] = dp[i-1][j-1] + dp[i-1][j]`。
- **实现细节**：
  - 直接用两层循环构造即可。
  - 时间复杂度 O(numRows^2)。
  - 空间复杂度 O(numRows^2) (存储结果)。

## 总结
这是一道基础的数学模拟题，也可以看作是简单的动态规划。
