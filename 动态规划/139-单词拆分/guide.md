# 139. 单词拆分

## 题目描述

给你一个字符串 `s` 和一个字符串列表 `wordDict` 作为字典。请你判断 `s` 是否可以利用字典中出现的单词拼接出。

**注意**：不要求字典中出现的单词全部都使用，并且字典中的单词可以重复使用。

**示例 1:**
```
输入: s = "leetcode", wordDict = ["leet", "code"]
输出: true
解释: 返回 true 因为 "leetcode" 可以由 "leet" 和 "code" 拼接成。
```

## 解题思路

### 方法：动态规划
- **状态定义**：`dp[i]` 表示字符串 `s` 的前 `i` 个字符（即 `s[0...i-1]`）能否被拆分成字典中的单词。
- **目标**：求 `dp[s.length()]`。
- **状态转移**：
  - 对于每个位置 `i`（从 1 到 n），我们需要枚举它的分割点 `j`（从 0 到 i-1）。
  - 如果 `dp[j]` 为 `true`（前 j 个字符能拆分），且子串 `s[j...i-1]` 在字典中存在。
  - 则 `dp[i] = true`。
- **优化**：
  - 可以把 `wordDict` 放入 `HashSet` 以实现 O(1) 查找。
  - 内层循环 `j` 可以从 `i-1` 往前找，或者限制最大单词长度，减少无效判断。

## 复杂度
- 时间复杂度：O(N^2)，其中 N 是字符串长度。
- 空间复杂度：O(N)。
